ðŸ“¦ Product Endpoints

GET /api/products
Get all products (with pagination)
GET /api/products?page=1&limit=5
Get products with pagination
GET /api/products/search?q=laptop
Search products by query
GET /api/products/search?category=Electronics&minPrice=100&maxPrice=500
Search with filters
GET /api/products/category/Electronics
Get products by category
GET /api/products/categories/list
Get all categories with counts
GET /api/products/images
Get all product images organized by folder name
GET /api/products/blends
Get blend file information for UI
GET /api/products/blends/folders
Get list of all blend folder names (use this to find correct folder names)
GET /api/products/blends/:folderName
Get specific blend file by folder name (example: chess_set_4k.blend). Use /api/products/blends/folders to get available folder names.
GET /api/products/blends/:folderName/file
Serve actual blend file for virtual world (add ?download=true to force download)
GET /api/products/blends/:folderName/textures/:textureName
Serve individual texture image (example: chess_set_board_diff_4k.png). Use /api/products/blends/:folderName to get texture names.
